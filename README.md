|----------|----------|-------------|
| [capability-brief.md](capability-brief.md) | All federal agencies | Master capability brief — product overview, security posture, use cases |

## Usage

These are markdown documents designed to be:
1. Read directly on GitHub by government evaluators
2. Converted to PDF via the `publish-pdfs` workflow (coming soon)
3. Attached to RFI/RFQ responses and capability statements

## Classification

**UNCLASSIFIED** — No CUI, no FOUO. Safe for public distribution.
